Instructions

Visit Google Takeout to download your location history. To do this, click the "Select None" button, and then reselect "Location History". Make sure to export the data as a json file. Google will then send you a link to download your data.

1. Select File

Unzip the file you downloaded from Google Takeout and upload Location History.json. Then, click select. Parsing this file will usually take about 5 minutes, especially if the file size is very large. If the website hangs, please allow it to continue.
